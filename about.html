Our Mission

Haryana Bus Info exists to make public transport information in Haryana free, fast, and accessible — without requiring an app download, a sign-up, or navigating complex government portals.

Millions of people across Haryana — students, workers, farmers, families — depend on Haryana Roadways every single day. Finding out when the next bus leaves should not require standing at the bus stand asking strangers or calling friends. We built this so you can check from home before you leave.

How We Build the Database

An automated, multi-stage data pipeline keeps the timetable current

🌐
Step 1 — Automated Scraping
A multithreaded Python + Playwright scraper collects raw schedule data from Haryana Roadways sources. Stable MD5-based hashing ensures reliable deduplication across multiple scraping runs.
haryana_scraper.py
🔄
Step 2 — JSON to CSV Conversion
Raw scraped data is converted from JSON to structured CSV. Via stops are extracted and normalized. Departure times are standardized to zero-padded 24-hour HH:MM format.
json_to_csv.py
🧹
Step 3 — Deduplication Merge
A merge script compares records using normalized time logic to remove false duplicates introduced by formatting inconsistencies, while preserving all genuine unique schedule entries.
merge.py
🔤
Step 4 — Station Name Cleaning
Station names are audited and fuzzy-matched using rapidfuzz with a 3-tier classification system — exact matches auto-applied, high-confidence fuzzy matches auto-applied above threshold, ambiguous cases flagged for manual review.
step1_find_names.py · step2_apply_fixes.py
✅
Step 5 — Manual Review & Upload
Flagged records are reviewed manually. Verified data is bulk-uploaded via a secure WordPress admin tool into the live MySQL timetable database serving haryanabusinfo.in.
bus-insert.php (V14)

Frequently Asked Questions

Common questions about Haryana Bus Info

Is Haryana Bus Info affiliated with Haryana Government or Haryana Roadways?
No. This is an independent, community-maintained resource. We are not affiliated with Haryana Roadways, Haryana State Transport, or the Government of Haryana in any way.
How current is the bus schedule data?
We update the database regularly as Haryana Roadways changes its timetables. However, schedules can change without public notice — always verify critical timings at your local bus stand or the official Haryana Roadways portal.
My bus route is missing. How do I report it?
Use the contact form at haryanabusinfo.in to report missing routes or incorrect timings. We review all reports and apply corrections in the next scheduled update.
Is this website free to use?
Yes — completely free, no sign-up required, no app to download, no charges ever. It is optimised to load fast on any phone including 2G connections, because many Haryana commuters are on low-bandwidth networks.
Can I use or republish this data?
Please contact us first via haryanabusinfo.in. We support public interest and research uses and are open to data partnerships for legitimate non-commercial applications.

About Haryana Roadways

Haryana Roadways, officially the Haryana State Transport, is the government-owned public bus service of Haryana, India. It operates an extensive network connecting all 22 districts of Haryana with major cities in neighbouring states including Delhi, Chandigarh, Rajasthan, Uttar Pradesh, and Punjab. The fleet comprises Ordinary buses for local routes, Express buses for inter-district travel, and Volvo AC coaches for premium long-distance services.

Why We Built This

Public bus timetable data in India is notoriously hard to find, fragmented across official portals, or simply unavailable online in a usable format. We built haryanabusinfo.in to solve this for Haryana — collecting, cleaning, and publishing the complete Haryana Roadways schedule in a searchable, mobile-friendly format. This companion resource on Cloudflare Pages extends that information to even more commuters.

Technology Behind the Site

The main timetable database at haryanabusinfo.in is built on WordPress with a custom PHP plugin and MySQL database. Data is collected using a Python + Playwright scraper, cleaned with fuzzy matching via the rapidfuzz library, and uploaded through a secure admin interface. The data pipeline processes thousands of route records through multi-stage deduplication and name normalization before any entry goes live.
